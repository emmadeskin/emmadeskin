
# Hi, I'm Emma Deskin 👋

Biomedical Engineering graduate from the University of Arizona (Cum Laude, May 2026).

**Programming:** MATLAB · Python · C · R · Arduino IDE
**Hardware:** Teensy · Raspberry Pi · Arduino · Stepper motors · Sensors · PCB assembly
**Software:** SolidWorks · ANSYS Fluent · Verasonics Vantage · ImageJ · Meshmixer · EAGLE CAD
**Skills:** Signal processing · V&V testing · 3D printing · Laser cutting · Technical documentation
